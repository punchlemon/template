## :eyes: Overview
> I am creating a library in C that works like the Go language.
> The functionality is very small, but the goal is to be able to use string, rune, slice, etc.

## :gun: Usage
```
git clone git@github.com:punchlemon/template
```
```
make ; ./template
```
